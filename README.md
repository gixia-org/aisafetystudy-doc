## 共建流程

建议参考以下流程开展活动：

1. 在Issue中发起活动倡议。
2. 确定主持人和时间后，在微信群和各个平台招募参与者。
3. 活动后，将相关的文档、视频、代码整理在MarkDown文件中，并提交至本仓库。

借助此流程，我们可以有效共建AI安全社区，并沉淀有价值的学习资源。
